# Ecommerce Mobile Homepage (WordPress Theme)

This is a custom WordPress mobile-first theme built for the assignment.
It includes:

✔ Mobile-only UI (430px phone view)
✔ Custom ACF fields (Hero banner, Brands, New Arrivals)
✔ WooCommerce mobile product grid (2-column)
✔ Hamburger drawer menu inside mobile-app-wrapper
✔ Clean responsive structure for assignment requirement

## Installation Steps
1. Install WordPress locally (XAMPP).
2. Clone or download this repository.
3. Place folder in: /wp-content/themes/
4. Activate theme from Admin → Appearance → Themes
5. Install required plugins:
   - Advanced Custom Fields (ACF)
   - WooCommerce
6. Import sample content (optional).
